# 概述
## 建立该项目的原因
 - 在嵌入式开发中，已有的C库很多时候并不适合嵌入式,特别是单片机应用,每次使用起来都很别扭
 - 每次开发中总会用到相同的接口或代码,为了最好的重复使用并减少BUG,需要整理和规划代码
 - 方便大家交流使用
 ## 内容概览
 
 -  common 通用的数据结构、系统和驱动接口等
 -  algorithm 算法相关
 -  device 设备驱动相关
 -  network 网络协议相关
 -  protocol 应用类协议相关
 -  misc 杂项
